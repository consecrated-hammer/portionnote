# Portion Note

Portion Note is a friendly, mobile-first tracker for meals, steps, and weight. It is built for quick logging and a clear view of how your day is going, without the clutter.

## What it does

- Log meals fast with minimal typing and big tap targets
- Track calories, protein, and net calories in one place
- Add steps and weight for any day, not just today
- Save favourite foods and reuse meal templates
- Get simple, practical suggestion nudges
- AI assisted food lookup and portion options when the database has no match
- Shared food library across users with owner-only editing
- View weekly summaries Monday to Sunday

## What to expect

- Clean, calm interface designed for phones first
- Quick add actions for meals, steps, and weight
- Food search with AI fallback when needed
- History view for reviewing and editing past days

## Core screens

- Today: Progress, remaining, quick add
- Log: Full day logging and multi item meals
- Foods: Library, favourites, recents
- Insights: Weekly summary and charts
- Settings: Targets, factors, export and import
